Azioni

#sviluppo-consapevole #tecnologie-democratiche #software-libero #conoscenza-libera #standard-aperti #open-source #libertà-digitali #digital-freedom

  1. Aprire e divulgare in Open Data i dati degli enti pubblici in modo che le aziende possano sviluppare servizi.
  2. Sostenere infrastrutture cloud di proprietà ed italiane. Esporre servizi ben documentati per sviluppatori;
  3. Richiedere ai fornitori il rilascio della documentazione per sviluppatore (API del software) per integrare diversi servizi;
  4. Prediligere soluzioni di software libero per la PA come previsto dal Codice di Amministrazione Digitale (CAD);
  5. Avviare percorsi per la definizione di standard territoriali attraverso il modello request for comment (RFC) che è alla base di Internet.
